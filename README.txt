Add a widget to display an icon or image with title and description on a widget area.

== Description ==

This plugin allows you to insert an image or an icon and below add a title and/or a text description.

You can see a live demo in [Made in Business theme](https://madeinthemes.com/demo/?theme=business).

== Installation ==

1. Upload `made-in-icon-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
